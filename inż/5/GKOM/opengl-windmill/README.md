# Building on Linux

From project root directory run:

```
make
```

Files from *src* folder will be compiled, their object files will be put into *obj* folder and, after linking them together, the resulting executable will be in *bin* folder

# Usage on Linux

From project root directory run:

```
./bin/Wiatrak
```
